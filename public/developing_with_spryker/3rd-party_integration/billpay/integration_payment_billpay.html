Payment Integration - Billpay (BETA)

Billpay is one of the leading payment providers in the European market. Our Billapy offers integration with this service.

Prerequisites

To use Billpay, you will need to create a Billpay account or use your existing credentials.

If you do not have an Billpay account, you can create one on the Billpay website:

https://www.billpay.de/en/business-clients/merchant-request/

You will also need your Merchant ID, Portal ID and security key - all provided by Billpay.

Additional important information can be found in the Billpay documentation - https://www.billpay.de/en/shopintegration/techdocs/


In order to integrate Billpay make sure you familiar with concept of Extending Spryker .

Installation

Composer dependency:

To install the , use composer, https://getcomposer.org/:

composer require spryker/billpay

Billpay visuals:

If you want to use Billpay visuals on your website please download them directly from Billpay https://www.billpay.de/en/shopintegration/downloads/
